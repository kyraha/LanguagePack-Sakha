Требуются разработчики
======================

Этот проект заброшен ввиду конфликта приоритетов начального разработчика, и давно не обновлялся.
Если у вас есть желание и возможности продолжить поддержку проекта, вы легко можете
создать "форк" данного репозитория и добавить ваши улучшения.
Проект открытый, под свободной лицензией Apache 2.0, бесплатно.

LanguagePack
